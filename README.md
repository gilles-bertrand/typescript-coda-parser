# Typescript coda parser

Parse a coda document and returns an object with the content of the document.
0 dependencies.

## Requirements

- Typescript >= 4.5.x
- node >= 16.x
- pnpm

## Install

```bash
pnpm i @triptyk/coda-parser
```

## Local Development

### Start

#### Dev

```bash
pnpm start:dev
```

### Compile

You need to transpile the Typescript before executing the command.

```bash
pnpm tsc
```

### Tests

```bash
pnpm test
```

### Lint

```bash
pnpm lint
```
